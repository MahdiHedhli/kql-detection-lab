# incident-triage

Reusable pivots to expand a single alert or identity into entities and timeline context fast.

- `alert-entity-expansion.kql`: expand one alert into its associated evidence set. ATT&CK: `triage pivot`
- `user-device-mail-timeline-pivot.kql`: pivot one user across identity, device, and mail telemetry. ATT&CK: `triage pivot`
